# claudemods Arch Linux Repositories

Professional Arch Linux repositories providing custom kernels and KDE development packages.

## Quick Installation

Add the following configuration to your `/etc/pacman.conf`:

```ini
[claudemods-v1]
SigLevel = Never
Server = https://claudemodsreloaded.co.uk/claudemods-v1/

[claudemods-dev]
SigLevel = Never
Server = https://claudemodsreloaded.co.uk/claudemods-dev/
